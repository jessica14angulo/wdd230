Repetition

BBC News

https://www.bbc.com/

The repetition principle is applicated on this page because the design repeats the same navigation bar, colors (Red, black, grey and white) in the entire page. It also uses the same fonts and, as we can see, the BBC web site is filled with the same distribution all over the page.

Proximity

Discovery

https://www.tudiscovery.com/

The proximity principle is applicated in every section of Discovery web page. As we can see, every topic that is exposed into this page has a name, a different color and, most important, are closer to each other. This helps us to differentiate every topic from this webpage.

Contrast

National Geographic

https://www.nationalgeographic.com/

The contrast principle is clearly seen in National Geographic web page. The color palette is filled with black, white and yellow, which are totally opposite colors. We can see the contrast between black and white and also between black and yellow. This helps us to clearly distinguish the text and also the images, which can highlight because of the black background.
